# Inline PDF Content Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Show imported PDFs inline in the document `Content` tab by default, with a Settings > Transcription toggle to show extracted PDF text only.

**Architecture:** Keep imported PDF storage unchanged: `Meeting.audioPath` remains the original PDF path and `Meeting.transcript` remains extracted text for summary/chat/RAG. Add a persisted `AppSettings.showExtractedPdfTextOnly` preference, route `MeetingDetailScreen` content rendering through that setting, and use Syncfusion's PDF viewer for the inline path.

**Tech Stack:** Flutter, Riverpod, SharedPreferences-backed `AppSettings`, Flutter localization ARB/generated classes, `syncfusion_flutter_pdfviewer`, Flutter widget tests.

---

  Widget _buildDocumentContentTab(
    Meeting meeting,
    MeetingNotifier provider,
    AppLocalizations l10n,
  ) {
    final settings = ref.watch(settingsProvider);
    final showPdfViewer = _isPdfDocument(meeting) &&
        !settings.showExtractedPdfTextOnly;

    return Column(
      children: [
        MaterialBanner(

          actions: const [SizedBox.shrink()],
          backgroundColor:
              Theme.of(context).colorScheme.surfaceContainerHighest,

        Expanded(
          child: showPdfViewer
              ? _buildPdfContentPane(meeting, l10n)
              : _buildExtractedDocumentTextPane(meeting, provider, l10n),
        ),
      ],
    );
  }

  Widget _buildPdfContentPane(Meeting meeting, AppLocalizations l10n) {
    final path = meeting.audioPath;
    if (path.isEmpty || !File(path).existsSync()) {
      return Center(
        child: Padding(
          padding: const EdgeInsets.all(24),
          child: Text(
            l10n.meetingDetailPdfFileMissing,
            textAlign: TextAlign.center,
          ),
        ),
      );
    }

    return SfPdfViewer.file(File(path));
