# summsumm

AI Text Summarizer — Android app for AI-powered text summarization via share-menu and text-selection.

## Features

- **Share-menu summarization**: Select text in any app and share to summsumm for instant AI summaries
- **Dual AI providers**: OpenRouter or OpenAI integration
- **Text-to-Speech**: Listen to summaries with TTS support
- **Fact Check mode**: Verify claims with investigative journalist AI prompt

## Setup

1. Configure API keys in app settings
2. For OpenRouter: Get key from [openrouter.ai](https://openrouter.ai)
3. For OpenAI: Get key from [platform.openai.com](https://platform.openai.com)

## Build

```bash
flutter build apk --release
```

- Flutter + Riverpod
